# Neon Digital Clock

A modern and responsive Digital Clock built using HTML, CSS, and JavaScript.

## Features

- Live Digital Clock (12-hour format)
- AM / PM Indicator
- Current Date Display
- Day of the Week
- New Year Countdown Timer
- Neon UI Design
- Responsive Layout
- Smooth Animations

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download the project files.
2. Open `index.html` in any web browser.

## Project Preview

This project displays:
- Current Time
- Current Date
- Day Name
- Countdown to New Year
